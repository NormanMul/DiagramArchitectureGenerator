# Genesis — Azure Architecture Diagram Generator

> _Democratize cloud architecture._

An AI-powered, agentic generator of Azure architecture diagrams that:

- Takes a **natural-language workload description** ("hub-spoke landing zone with AKS workload, Azure SQL MI, and Front Door fronting App Gateway").
- Picks the closest matching **reference architecture** from the Azure Architecture Center catalog.
- Produces a syntactically correct **Python `diagrams` script**, an editable **`.drawio`** export, and an inline **SVG/PNG**.
- Uses **unmodified official Microsoft Azure icons (V19)**.
- Lets you **iterate** via chat ("add a private endpoint to SQL").

> **Status**: scaffold in progress (Phase 1). See [docs/architecture.md](docs/architecture.md) for the target system and [docs/prompts.md](docs/prompts.md) for the full build spec.

## Quickstart (developer)

### Prerequisites

| Tool | Version | Install |
|---|---|---|
| Python | 3.12 | https://www.python.org/ |
| uv | ≥ 0.11 | `pip install uv` or [official installer](https://docs.astral.sh/uv/) |
| Node.js | 24 LTS | https://nodejs.org/ |
| pnpm | ≥ 10 | `corepack enable && corepack prepare pnpm@latest --activate` |
| Azure CLI | ≥ 2.85 | https://aka.ms/installazurecli |
| Bicep CLI | ≥ 0.43 | `az bicep install` |
| GitHub CLI | ≥ 2.87 | https://cli.github.com/ |
| Graphviz | latest | Required by the renderer (`brew install graphviz` / `apt install graphviz` / `choco install graphviz`) |

### Local dev

```bash
# Backend
cd backend
uv sync
uv run uvicorn app.main:app --reload --port 8000

# Frontend (separate shell)
cd frontend
pnpm install
pnpm dev   # http://localhost:3000

# Download Azure Icons V19 (one-time, before first render)
pwsh ./scripts/download-azure-icons.ps1
```

### Infra what-if (no deploy)

```bash
az login
az account set --subscription "ME-MngEnvMCAP818878-mprawironego-1"
az deployment sub what-if \
  --location southeastasia \
  --template-file infra/main.bicep \
  --parameters infra/parameters/prod.bicepparam
```

Deployment is gated through GitHub Actions with OIDC; no service principal secrets in this repo.
