## Microsoft Azure Architecture Icons (V19)

**Source**: https://learn.microsoft.com/azure/architecture/icons/
**Bundled location**: `icons/azure_V19/`
**Version**: V19 (downloaded via `scripts/download-azure-icons.ps1`)
**License**: Microsoft Azure Architecture Icons Terms of Use (bundled at
`icons/azure_V19/Terms_of_Use.pdf`)

Key constraints we comply with (see Terms of Use for full text):
- Icons are bundled **as downloaded** with **no modification** (no recoloring,
  rotation, flipping, cropping, or restyling).
- The renderer (`backend/app/renderer/icon_catalog.py`) programmatically rejects
  any SVG mutation (`fill`, `transform`, `filter` attributes set on icon nodes).
- Icons are used only as components of architecture diagrams generated by this
  tool — never offered for individual download.
- The UI footer surfaces an attribution link to Microsoft's icon terms.
- A monthly GitHub Action (`.github/workflows/refresh-icons.yml`) diffs the
  bundled set against the latest V-N pack on Microsoft Learn and opens a PR
  when a new version is published.

If you fork or redistribute this project, you must continue to comply with
the Microsoft Azure Architecture Icons Terms of Use.
